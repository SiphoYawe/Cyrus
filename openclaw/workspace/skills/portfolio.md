# Portfolio Skill

## Trigger Phrases
- "show my portfolio"
- "what's my balance"
- "how much do I have"
- "portfolio overview"
- "chain allocation"

## Workflow

1. Call `portfolio` tool (optionally with chain filter)
2. Display total USD value prominently
3. Show chain-by-chain breakdown with percentages
4. Mention active in-flight transfers if any
5. Suggest actions if portfolio is imbalanced

## Response Template

**Portfolio Value: $X,XXX.XX**

| Chain | Value | Allocation |
|-------|-------|-----------|
| Ethereum | $X,XXX | XX% |
| Arbitrum | $X,XXX | XX% |
